# Rzopierdol Esport Strats

Statyczna strona pod taktyki CS dla Rzopierdol Esport.

## Upload bez instalowania

1. Wejdz w repo: https://github.com/wertPL/rzopierdol-esport-strats
2. Kliknij `Add file`.
3. Kliknij `Upload files`.
4. Wrzuƒá `index.html`, `README.md` oraz folder `assets`.
5. Kliknij `Commit changes`.

## GitHub Pages

1. Wejdz w `Settings`.
2. Otworz `Pages`.
3. Wybierz `Deploy from a branch`.
4. Ustaw branch `main` i folder `/ root`.
5. Kliknij `Save`.

Po chwili strona bedzie dostepna pod:

https://wertpl.github.io/rzopierdol-esport-strats/

## Edycja taktyk

Na razie dane sa wpisane w `index.html` w zmiennej `strats`. Po podeslaniu linkow do Google Docs mozna podmienic placeholdery na realne mapy, kategorie i rozpiski.
